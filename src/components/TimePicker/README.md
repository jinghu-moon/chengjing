# TimePicker 时间选择器

## 概述

轻量级时间选择器组件，仅包含时和分两个字段。通过滚轮滚动或直接输入调节时间值，数字切换带有方向感知的滚动动画。

## 目录结构

```
TimePicker/
└── index.vue    # 单文件组件（逻辑 + 模板 + 样式）
```

## Props

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `modelValue` | `string` | `'12:00'` | 时间值 (v-model)，格式 `HH:mm` |

## 事件

| 事件 | 参数 | 说明 |
|------|------|------|
| `update:modelValue` | `string` | 时间值变更，格式 `HH:mm` |

## 交互方式

| 操作 | 行为 |
|------|------|
| **滚轮滚动** | 在数字区域滚动调节值，累积阈值 30px 触发一次变更 |
| **点击输入** | 聚焦后全选，直接键入数字，失焦时 clamp 到合法范围 |
| **Enter 键** | 确认输入并失焦 |

## 滚轮逻辑

- **累积器机制**：`accHour` / `accMin` 累积滚轮 deltaY，超过 `WHEEL_THRESHOLD`（30px）才触发一次调节
- **deltaMode 适配**：`deltaMode === 1`（行模式）时乘以 20 转换为像素
- **方向感知**：记录 `hourDirection` / `minuteDirection`（`up` / `down`），驱动滚动动画方向
- **循环溢出**：小时 0↔23，分钟 0↔59

## 滚动动画

数字切换使用 Vue `<Transition>` + `:key` 触发：

| 动画名称 | 效果 |
|----------|------|
| `scroll-up` | 新数字从下方滑入，旧数字向上滑出 |
| `scroll-down` | 新数字从上方滑入，旧数字向下滑出 |

动画时长 300ms，缓动 `cubic-bezier(0.25, 0.46, 0.45, 0.94)`。

## 与 DatePicker 的区别

| 特性 | TimePicker | DatePicker/TimePanel |
|------|-----------|---------------------|
| 字段 | 时 + 分 | 时 + 分 + 秒 |
| 交互 | 滚轮累积 + 滚动动画 | 滚轮即时 + Tab 导航 |
| 定位 | 内联渲染 | 嵌入 Teleport 面板 |
| 数据格式 | `HH:mm` 字符串 | `defineModel` 数值 |
